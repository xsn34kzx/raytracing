## Description

Originally an assignment based on Peter Shirley's [Ray Tracing in One
Weekend][raytracing-git] completed for NYU's [Special Topics: Computer Graphics
(CSCI-UA 480-041)](https://cs.nyu.edu/courses/spring23/CSCI-UA.0480-041/)
during the Spring 2023 semester, this project aims to rewrite and expand that
assignment, primarily for a deeper understanding of ray tracing techniques and
programming language exploration.
